# Studio HB FIT - Website Oficial ✨

> Website institucional, responsivo e de alta performance desenvolvido para o Studio HB FIT (Pilates Funcional), localizado em Florianópolis - SC.

Este projeto foi construído do zero utilizando tecnologias web nativas, focado em uma experiência de usuário premium (UX), carregamento rápido, SEO otimizado e total 
responsividade para dispositivos móveis.

---

## 🚀 Funcionalidades (Features)

*   **Design Totalmente Responsivo:** Adaptação fluida para smartphones, tablets e computadores de alta resolução.
*   **Menu Hambúrguer Customizado:** Navegação mobile nativa em formato de gaveta (*navigation drawer*) com animações fluidas via CSS e JavaScript.
*   **Efeitos Visuais Avançados:**
    *   *Cursor Glow Effect:* Um brilho sutil que acompanha o ponteiro do mouse (desativado automaticamente em telas de toque para melhor performance).
    *   *Scroll Reveal:* Elementos surgem elegantemente na tela à medida que o usuário rola a página.
*   **Seções Institucionais Completas:** Sobre a empresa, Metodologia, Planos/Plataformas Parceiras (Wellhub, Totalpass, GoGood), Serviços, Equipe e Sublocação.
*   **Integrações Práticas:**
    *   Localização via mapa dinâmico do Google Maps.
    *   Botão flutuante e CTAs direcionando direto para o atendimento automatizado do WhatsApp.

---

## 🛠️ Tecnologias Utilizadas

O projeto priorizou a simplicidade arquitetural e a performance, utilizando a stack nativa da web (sem frameworks pesados):

*   **HTML5:** Estruturação semântica e acessível.
*   **CSS3:** Estilização moderna utilizando Variáveis CSS (Custom Properties), Flexbox, CSS Grid, e Media Queries para responsividade.
*   **JavaScript (ES6+):** Lógica de interações, manipulação do DOM para o menu mobile e controle dos efeitos de animação.
*   **Font Awesome:** Biblioteca de ícones vetoriais.
*   **Google Fonts:** Tipografia integrada (Manrope/Sans-serif).

💻 Como Rodar o Projeto Localmente
Como o projeto é composto por arquivos estáticos, você não precisa instalar nenhuma dependência (como Node.js ou npm).

Clone o repositório: (bash)
git clone [https://github.com/maludalsasso/sitepilateshbfit] (https://github.com/maludalsasso/sitepilateshbfit)

Navegue até a pasta do projeto: (bash)
cd studio-hbfit

Abra o arquivo principal:

Basta dar um duplo clique no arquivo index.html ou utilizar a extensão Live Server no VS Code para rodar com atualização em tempo real no endereço http://127.0.0.1:5500

🌐 Deploy e Hospedagem
